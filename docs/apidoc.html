Loshu.js reference

In loshu.js, a vector is expressed as a plain javascript array, e.g. [1,2,3], and a matrix is simply an array of arrays, in row-major format. e.g. [[1,2,3],[4,5,6],[7,8,9]].
the library follows a "functional" paradigm, i.e. functions do not modify inputs and objects do not have methods themselves.
all the functions in the library are under the namespace LOSHU, or lo in shorthand (e.g. lo.add(A,B)), though for the rest of this document the prefix will be ommited for conciseness.
some functions (marked with WASM ✔︎) are also implemented in C++ and ported to web assembly, for (around 100x) speed boosts. for how to enable wasm mode (and how to get started in general) please consult the README.

library options

options.assertionlvl
assertion level. one of ASSERTION_NONE, ASSERTION_WARN, ASSERTION_DEFAULT, ASSERTION_THROW. functions can optionally do basic typechecks on the input, and give different types of response (specified by this option) when a problem is found. set to ASSERTION_NONE to disable the feature (and optimise for speed). default is ASSERTION_DEFAULT
options.termcols
width of the terminal in columns. affects how many elements are shown when pretty printing matrices. default is 80
options.usewasm
whether or not to use Web Assembly acceleration. iff loshuwasm.js is detected, this will default to true

affine()

create an affine homogeneous transformation matrix from a selection of presets

syntax

affine('rot2d',theta)
2d rotation by theta radians
affine('rotx',theta)
3d rotation around x axis by theta radians
affine('roty',theta)
3d rotation around y axis by theta radians
affine('rotz',theta)
3d rotation around z axis by theta radians
affine('rotaxial',theta,[x,y,z])
3d rotation by theta radians around axis [x,y,z]
affine('transl2d',[x,y])
2d translation with offset [x,y]
affine('scale2d',[x,y])
2d scaling by scale factor x and y along the axes
affine('scale2d',x)
2d scaling by scale factor x along both axes
affine('scale3d',[x,y,z])
3d scaling by scale factor x, y and z along the axes
affine('scale3d',x)
3d scaling by scale factor x along all three axes
affine('reflect2d',[x,y])
2d reflection about axis [x,y]
affine('reflect3d',[x,y,z])
3d reflection about axis [x,y,z]

returns

the transformation matrix (homogeneous, i.e. if transform is 2d, this will be 3x3; if transform is 3d, this will be 4x4)

reference

  • https://en.wikipedia.org/wiki/Affine_transformation

    dist()

    euclidean distance between points, lines, and planes in n-dimensional space.
    • a point is denoted by the vector [x1,x2,...,xn]
    • a line is denoted by [o,d], where o=[x1,x2,...xn] is a point on the line, and d=[d1,d2,...dn] is the direction vector (i.e. equation: o + t d)
    • a plane is denoted by [a1,a2,...,an,d] where a1*x1+a2*x2+...+an*xn=d is the equation of the line

    syntax

    dist('pt',u,'pt',v)
    calculate distance between points u and v
    dist('pt',u,'ln',[o,d])
    calculate distance between point u and line o + t d
    dist('pt',u,'pl',[a1,a2,...an,d])
    calculate the distance between point u and plane a1*x1+a2*x2+...+an*xn=d
    dist('ln',[o,d],'pt',u)
    same as ('pt', u, 'ln', [o,d])
    dist('ln',[o1,d1],'ln',[o2,d2])
    calculate the distance between two lines
    dist('ln',[o,d],'pl',[a1,a2,...an,d])
    calculate the distance between a line and a plane
    dist('pl',[a1,a2,...an,d],'pt',u)
    same as ('pt', u, 'pl', [a1,a2,...an,d])
    dist('pl',[a1,a2,...an,d],'ln',[o,d])
    same as ('ln', [o,d], 'pl', [a1,a2,...an,d])
    dist('pl',[a1,a2,...an,d],'pl',[b1,b2,...bn,c])
    calculate distance between two planes

    returns

    the distance (a number)

    gauss()

    gaussian elimination on a matrix

    syntax

    gauss(A,B,{options})
    perform gaussian elimination on the augmented matrix [A|B] to reduce A to row echelon form

    options

    jordan
    if true, additionally perform gauss-jordan elimination to produce a reduced row echelon form
    elementary
    if true, additionally calculate the elementary matrices from the row operations
    operators
    an array containing the types of elementary row operations to use: 's'(switching rows), '*' (multiplying by a constant) and '+*' (adding a multiple of another row)
    default:  {jordan:false,elementary:false,operations:['s','+*','*']}

    returns

    [A', B'] where [A'|B'] is the resultant matrix. if option elementary is set to true, returns [A', B', [E0, E1, E2, ...]] where En is an elementary matrix

    reference

    • https://en.wikipedia.org/wiki/Gaussian_elimination

    see also
